README

This directory contains Microsoft Visual Studio v15 (2017) solution
files.

The Solution assumes a directory setup as follows:

Projects/         	Location of Visual Studio projects.
-dragonfly/       	Source code for game.
--vs-2017/        	Visual Studio solution files.
---Debug/         	Location of .exe that run in Debug mode.
---include/	  	Location of dragonfly header files
----saucershoot/	Location of saucershoot example headers
---lib/			Location of dragonfly cpp files
----saucershoot		Location of saucershoot example cpp files
---Release/       	Location of .exe that run in Release mode.
--df-font.txt     	Fonts used by Dragonfly.
--df-config.txt   	Config file used by Dragonfly (optional).
--sounds/         	Sounds used for game.
--sprites/        	Sprites used for game.
-SFML-2.4/        	SFML directory (version 2.4+).
